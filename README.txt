Установите в ваш Python эти модули
pip install Flask
pip install dnspython


Весь DNS сервер работает из коробки, запускается файлом server.py.
База данных сервера, работает на sqlite3 в случае если вы в процессе эксперимента испортили базу данных,
либо хотите обнулить кеш, удалить списки разрешенных и забаненых, просто удалите файл dnsbase,
структура новой базы данных сформируется автоматически при новом запуске и будет чиста
